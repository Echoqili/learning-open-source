### 🚀 快速开始 | Quick Start

#### 方法 1: 克隆完整技能库

```bash
# 克隆 antigravity-awesome-skills (推荐)
git clone https://github.com/sickcn/antigravity-awesome-skills.git ~/.agent/skills

# 克隆 claude-skills
git clone https://github.com/alirezarezvani/claude-skills.git ~/.claude/skills
```

#### 方法 2: 使用 Claude Code 插件安装

```
/skill-install antigravity-awesome-skills
```

#### 方法 3: 单独安装特定技能

```bash
# 安装 TDD 技能
npx ai-agent-skills install sickcn/antigravity-awesome-skills/tdd

# 安装架构模式技能
npx ai-agent-skills install sickcn/antigravity-awesome-skills/architecture-patterns
```
